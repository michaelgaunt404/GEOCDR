WELCOME TO GeocodeR

An interactive, web-based tool to help you with all your geocoding needs.

GeocodeR enables intuitive and rapid geocoding.

 WHAT YOU CAN DO WITH GeocodeR:

  • Find coordinates for partial or full addresses.
  • Find addresses for coordinates. 
  • Review both via interactive maps. 

 WHAT YOU NEED BEFORE YOU USE GeocodeR:

  • A personal API key, get one form the HERE website.
    • Create a freemium project and you will be provided a REST api key - this is your secret password required to use GeocodeR. 
    • Freemium projects cap out at 250K geocode transactions per month. 
    • Each address or coordinate row in your uploaded file will count as one transaction 
    • You can check that you have successfully loaded your data before you geocode via the Uploaded Data tab.
  • A list of addresses or coordinates which you want to geocode. 
    • Click on the relevant tab to perform your geocoding operation - Addresses to convert addresses to coordinates and Coordinates to convert coordinates to addresses. 
    • Addresses must be contained in a single file with a single addresses column. 
    • Coordinates must be defined in your data by individual lat/long columns - don't abbreviate with a trailing period (eg. lat.) or in it's entirety. 
    • This application strictly uses the ESPG:4326 cooridante reference system for mapping purposes.
